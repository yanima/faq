设置打印机
打印概述
常见问题

  • 1.HP LaserJet Pro M226dw

    • 步骤一： 将HP微信打印机连入网络

      注意：此产品仅适用于家用网络环境，在企业网络环境下可能会造成连接失败。若有防火墙以及杀毒软件的一些限制，请咨询企业IT。

    • 方法一：有线连接

      1. 将网线插入打印机的后侧网路接口。

      2. 等待打印机建立网络连接。在打印机控制面板上，点击“网络” 按钮。如果已列出有效的 IP 地址，则表示网络连接已成功。IP 地址示例：192.168.0.1

    • 方法二：无线连接

      1. 点击打印机右前方控制面板“无线”( ) ->点击“无线功能”->点击“打开/关闭无线功能”点击->“打开”开启打印机无线功能

      2.在打印机控制面板上点击“无线” 按钮，然后点击“无线设置向导”。

      3.选择无线热点，如果需要密码，输入密码。等待打印机建立网络连接。在打印机控制面板上，点击“网络” 按钮。如果已列出 IP 地址，则表示网络连接已成功。

    • 步骤二： 开启HP web 服务

      点击控制面板顶部的 “Web 服务设置图标”->点击 “启用Web服务”->点击 “OK”->再次点击 “OK” 确认Web 服务已经完成->最后点击 “打印信息页表”。等待打印机信息页打印完成，这可能需要几分钟。

  • 2.HP LaserJet Pro MFP M227fdw / M230fdw

    • 步骤一： 将HP微信打印机连入网络

      注意：此产品仅适用于家用网络环境，在企业网络环境下可能会造成连接失败。若有防火墙以及杀毒软件的一些限制，请咨询企业IT。

    • 方法一：有线连接

      1.将网线插入打印机的后侧网路接口。

      2.等待打印机建立网络连接，在打印机控制面板上，点击右上角的“网络”( )，再点击“网络已连接”，如果已经列出有效的IP地址，如下图的：IPv4: 15.96.153.57，则表示网络已经成功连接。

    • 方法二：无线连接

      1.点击打印机控制面板“无线”( ) ->点击“网络Wi-Fi关”->点击左下角“设置”->点击“开/关无线功能”->点击“开” 。开启打印机无线功能。

      2.在打印机控制面板上点击“无线” 按钮，然后点击“网络Wi-Fi开”->点击左下角“设置”->点击“无线设置向导”。

      3.选择无线热点，如果需要密码，输入密码，等待打印机建立网络连接。在打印机控制面板上，点击“网络Wi-Fi开” 按钮。如果已经列出IP地址，如IPv4:172.20.10.5，则显示无线网络连接已经成功。

    • 步骤二： 开启HP web 服务

      滑动控制面板到最后一页，点击控制面板的“设置”->点击“HP Web 服务”->点击“启用 Web 服务”->点击“OK”->再次点击“OK” 确认Web 服务已经完成->最后点击“打印信息页表”。

      以下图文为：如何开启打印机的Web服务（HP LaserJet Pro MFP M227fdw）

      1.获取二维码

      接入网络，按图示步骤开启Web服务，打印信息页。

      2. 等待打印机信息页打印完成，这可能需要几分钟。

  • 3.HP OfficeJet Pro 8710 / 6960 / 6970

    • 步骤一： 将HP微信打印机连入网络

      注意：此产品仅适用于家用网络环境，在企业网络环境下可能会造成连接失败。若有防火墙以及杀毒软件的一些限制，请咨询企业IT。

    • 方法一：有线连接

      1.将网线插入打印机的后侧网路接口。

      2.等待打印机建立网络连接。从屏幕顶端，触摸并向下滑动打开控制面板，点击顶部左起第二个“以太网Ethernet”图标。如果已列出有效的 IP 地址，则表示网络连接已成功。IP 地址示例：192.168.0.1

    • 方法二：无线连接

      1.从屏幕顶端，触摸并向下滑动打开控制面板->点击“设置”

      2.点击“网络设置”->点击“无线设置”->开启打印机无线功能->点击“无线设置向导”。

      3.选择无线热点，如果需要密码，输入密码。点击OK连接。如果已列出 IP 地址，则表示网络连接已成功。

    • 步骤二： 开启HP web 服务

      点击控制面板顶部的 “Web 服务设置图标”->点击 “启用Web服务”->点击 “OK”->再次点击 “OK” 确认Web 服务已经完成->最后点击 “打印信息页表”。等待打印机信息页打印完成，这可能需要几分钟。

  • 4.HP DeskJet 3636 IA/ 3638 IA

    • 步骤一：打印机硬件安装

      注意：请勿在此步骤连接USB线

      请参阅箱内安装海报完成下列步骤：

      1. 拆开包装
      2. 打开电源
      3. 安装墨盒
      4. 装入普通白纸
      安装过程中可能会打印出打印校准页，请按照页面说明完成打印校准。

    • 步骤二：安装驱动并接入网络

      注意：此产品仅适用于家用网络环境，在企业网络环境下可能会造成连接失败。若有防火墙以及杀毒软件的一些限制，请咨询企业。请根据您的需要选择通过计算机或移动设备进行连接

    • 方法一：计算机连接

      1.将 HP 打印机软件CD插入计算机中，或在Web浏览器中输入123.hp.com/dj3638下载驱动程序。安装打印机驱动程序时，请选择“无线连接”方式，以正常使用微信打印功能。

    • 方法二：移动设备连接

      1.点击123.hp.com/dj3638下载并打开 HP All--One Printer Remote app

      *部分Android系统手机需要下载并安装“HP Print Service Plugin”插件。

      2.请按照app所述步骤完成安装，确保完成无线网络连接，当您看到如下页面，代表连接成功

      3.若没有成功出现上述页面，请按如下路径，接受使用条款后，手动激活网络服务：

      1. iOS: App主页 “打印机设置” “高级设置” Web服务 “设置”。
      2. Andorid: App主页右上角菜单 “打印机设置” Web服务 “设置”。

    • 步骤三：注册HP Connected帐号（可选）

      忽略此步骤不影响后续惠普微信打印使用。

    • 步骤四：设置HP QQ打印机

      注意：请确保您已完成前述步骤1、2，且打印机控制面板无线指示灯与ePrint指示灯常亮。

      注意：当完成以上步骤，将自动打印出一张带二维码的信息页。若打印机无法打印二维码，请同时按下打印机控制面板上的ePrint和信息按钮。
